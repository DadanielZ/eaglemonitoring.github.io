MODULES

Eagle is built using big data solutions to scale for big data use cases.

Data Stream Processing

Eagle listens and processes large metrics and event stream in real time.

Policy Engine

Eagle evalutes the pre-defined policy with complex business logic on the fly.

Machine Learning Engine

Eagle predicts anomalies via a built-in machine learning engine

Remediation Engine

Eagle integrates with 3rd parties applications like Apache Ranger, Hadoop Robot, etc to trigger auto remediation workflow.

Data Storage

Eagle supports HBase, Durid and Elastic Search as data storage by default. RDBMS like MySQL or Oracle are supported with JDBC.

Generic Dashboard

Eagle provides a customized generic dashboard to allow user to visulaze their metrics with historical trend and drill down capabilities.
