ABOUT EAGLE

Apache Eagle is an Open Source monitoring solution for Hadoop to instantly identify access to sensitive data, recognize attacks and malicious activities in hadoop and block access in real time.

Eagle secures Hadoop in realtime in 3 steps:

Step 1

Listens to the various user generated signals by streaming through the Hadoop Audit and System logs.

Step 2

Analyzes these signals to detect anomalies based on predefined security policies and machine learning models according to the user's past behavior.

Step 3

Generates Alerts on detection of anomalies and remediates user activity/access if necessary.

Additional Bonus

Eagle also provides an extensible architecture and framework for developers to easily onboard new use cases into the framework.

MODULES

Eagle is built using big data open source products to scale for big data use cases.

Realtime data stream processing engine

Eagle listens and processes various logs and access patterns in real-time using open source big data products like STORM, Kafka, Hbase and Druid.

Policy Manager

Expressive language - Users can create and modify policies for alerting and remediation on certain data activity monitoring events.

Machine learning Models

1. Anomaly detection

2. User profiling - Eagle creates user profiles consisting of multiple dimensions. These user profiles automatically help build the policies for monitoring and alerting.

Data classification

You can import data classification standards and policies from tools like dataguise dg secure and apache ranger.

Remediation

Eagle provides the ability to detect and stop a threat. For example, If a malicious users is trying to copy data from the hadoop cluster to local system eagle alerts the admins and also stops the user from copying the data.

User activity Exploration

Eagle provides the ability to drill down into alert details to understand the data security threat. It provides a simple intuitive user interface to help analyze the threat to a greater detailed extent.

USE CASES

Data loss prevention

Get alerted and stop a malicious user trying to copy, delete, move sensitive data from the hadoop cluster.

Malicious Logins

Detect login when malicious user tries to guess password. Eagle creates user profiles using machine learning algorithm to detect anomalies. This anomaly detection together with the policy for user logins would trigger an alert and block this user from accessing sensitive datasets.

Unauthorized access

Detect and stop a malicious user trying to access classified data without privilege. Unauthorized access is one fact of Eagle user profiles, and machine learning algorithm will detect anomaly. This anomaly detection together with the policy of user access to classified data without authorization would trigger an alert to user's manager.

Malicious user operation

Detect and stop a malicious user trying to delete large amount of data. Operation type is one parameter of Eagle user profiles. Eagle supports multiple native operation types.
