MODULES

Eagle is built using big data open source products to scale for big data use cases.

Realtime data stream processing engine

Eagle listens and processes various logs and access patterns in real-time using open source big data products like STORM, Kafka, Hbase and Druid.

Policy Manager

Expressive language - Users can create and modify policies for alerting and remediation on certain data activity monitoring events.

Machine learning Models

1. Anomaly detection

2. User profiling - Eagle creates user profiles consisting of multiple dimensions. These user profiles automatically help build the policies for monitoring and alerting.

Data classification

You can import data classification standards and policies from tools like dataguise dg secure and apache ranger.

Remediation

Eagle provides the ability to detect and stop a threat. For example, If a malicious users is trying to copy data from the hadoop cluster to local system eagle alerts the admins and also stops the user from copying the data.

User activity Exploration

Eagle provides the ability to drill down into alert details to understand the data security threat. It provides a simple intuitive user interface to help analyze the threat to a greater detailed extent.
